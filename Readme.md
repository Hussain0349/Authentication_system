# Node.js Task Management System

A simple and efficient Task Management System built with Node.js, allowing users to register, log in, log out, and manage their tasks. This project demonstrates basic user authentication and CRUD operations for tasks using a server-side JavaScript environment.

## Features

### User Authentication

- Register new users with secure password storage.

- Log in existing users with session or token-based authentication.

- Log out users securely.

- Task Management

- Create new tasks.

- View all tasks assigned to a user.

- Update existing tasks.

- Delete tasks when completed or unnecessary.

- Security

- Passwords are hashed before storing in the database.

- Sensitive information like .env is not uploaded to GitHub.

### Technology Stack

- Node.js

- Express.js

- MongoDB (or any preferred database)

- Mongoose ORM

- bcrypt for password hashing

- JSON Web Tokens (JWT) for authentication
